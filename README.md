📖 模組詳細介紹 (Description)
在《殺戮之塔 2》的多人連線環境中，如果兩位玩家安裝的角色 Mod 順序不一致，BaseLib 生成的房間 Hash 就會產生偏移，導致系統在進入房間前直接將隊友阻斷。
以往的解決方法需要玩家大費周章去翻找電腦的
AppData隱藏資料夾，用記事本手動修改
settings.save，非常麻煩。
ModListSorter就是為了終結這個繁瑣步驟而生的全自動工具！ 它會在遊戲關閉退場的瞬間，自動攔截系統存檔，並將你的模組清單進行重構：
BaseLib等核心前置庫強制置頂，確保底層加載安全。
其餘所有自訂角色與功能 Mod，一律強制按字母 A-Z 進行重新排序。
只要所有要一起連線的朋友都安裝了這個模組，大家的模組加載順序與房間 Hash 就能永久保持 100% 鏡像對齊！

🚀 安裝與使用說明 (How to Use)
為了讓排序生效，請務必按照以下簡單的「開-關-開」步驟操作：
下載與解壓：下載本模組的壓縮檔。
放入目錄：解壓縮後，將整個
ModListSorter資料夾直接丟進遊戲根目錄的
mods資料夾中。
第一次開啟：啟動遊戲進入主選單（此時模組已成功加載，並在背景準備就緒）。
關閉遊戲：直接退出遊戲（模組會在退場的這一秒，自動把你的設定檔排好 A-Z 順序）。
第二次開啟：重新啟動遊戲！大功告成！
此時你的模組順序已經跟所有安裝此工具的隊友完全一致，可以直接秒進房間，開啟快樂的多人連線大亂鬥！
