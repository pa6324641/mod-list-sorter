支援多人模式
1. 描述
本模組為《殺戮之塔 2》引入了全新角色——來自《九星毒奶》的江曉 (JiangXiao)。
江曉是一位以極深紅色為代表色的角色，他不是一個單純的戰士，而是一個透過不斷點亮「內視星圖」來獲得質變的成長型角色。隨著挑戰層數的深入，江曉的力量將不斷覺醒與突破。

3. 核心戰鬥機制
江曉的戰鬥體系圍繞著「星力等級」、「星技品質」以及特殊的「基礎技藝」展開：
🛡️ 內視星圖 (Inner Star Map) - 成長基石
這是江曉的核心遺物。透過戰鬥勝利（一般 +625、精英 +1250、首領 +2500）累積技能點。
動態進化： 技能點不僅是數值，它還關聯到江曉的「星力等級」與「星技品質」，決定了卡牌數值的強弱。
✨ 星力等級 (Star Power Level)
反映當前的修為境界：星塵期 → 星雲期 → 星河期 → 星海期 → 星空期 → 星盡期。
隨著境界提升，江曉會獲得額外的每回合能量，象徵著星力的充沛。
💎 星技品質 (Star Skill Quality)
卡牌威力會隨品質（黃銅、白銀、黃金等）提升。
許多卡牌（如「青芒」）的傷害或效果會標註「隨星技品質提升」，使同一張牌在後期比前期強大數倍。
⚔️ 基礎技藝 (Basic Arts)
包含徒手、刀、弓、匕首、長戟等。卡牌具有不同的技藝標籤（Tag），對應不同的等級加成。

4. 代表性卡牌分析
江曉的卡組展現了鮮明的「功能性攻擊」與「治療/增益」並存的特色：
【青芒】 (Qing Mang)： 核心輸出。造成傷害並減少目標力量；面對擁有「人工製品」的敵人時，傷害會暴增（5倍）。
【忍耐】 (Ruinance)： 防禦機制。獲得格擋的同時獲得「忍耐」能力，在戰鬥中持續累積優勢。
【祝福】 (Blessing)： 生存續航。罕見的治癒手段，與其「奶媽」的角色設定呼應。
【打擊/防禦】： 專屬版本，名稱與敘述都與江曉個人劇情掛鉤。
